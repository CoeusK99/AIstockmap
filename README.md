# 台灣科技股產業地圖

互動式的台灣科技股關係圖:把 **供應鏈(誰供應誰)**、**集團/持股** 與 **競爭關係**
畫成一張力導向網路圖,一眼看出台灣科技股之間的關聯性。

## 功能

- **產業鏈分層檢視(預設)**:八個欄位由左(上游:設備材料)到右(下游:品牌/海外終端),
  欄位標題固定在畫面頂端;欄內依子族群分區(載板PCB、散熱液冷、電力設備…),
  小標題可點擊摺疊/展開(摺疊成帶數量的氣泡);可切換「自由網絡」檢視
- **190 個節點,系統性涵蓋科技業與相關產業**:台股八大科技族群(IC 設計、晶圓製造、
  記憶體、封測、設備材料通路、關鍵零組件、組裝代工、品牌網通)+ 海外錨點
  (輝達、蘋果、雲端巨頭、低軌衛星巨頭、機器人應用端…),涵蓋矽晶圓與磊晶、功率半導體、
  被動元件、封裝/自動化設備、機器人與邊緣 AI、CPO 矽光子、特用化學與綠色循環、
  低軌衛星、軟體與資安、CCL、伺服器機構、測試介面、工業電腦等關聯產業
- **四種關係線**:供應(曲線箭頭,線色 = 供應方的產業色)、集團/持股(虛線)、
  互相持股(點劃線,如文曄⇄祥碩、聯電⇄頎邦)、競爭(點線),
  滑過線可看關係說明(例:台積電 → 輝達「AI GPU 先進製程獨家代工」);
  選取個股時,它的供應線會以流動動畫顯示方向
- **點擊個股**:右側面板列出上游供應商、下游客戶、集團與競爭對手,可一路點擊跳轉
- **篩選**:產業別圖例點擊篩選、題材標籤(AI/伺服器/蘋果鏈/手機/網通)、關係線開關
- **AI 供應鏈十大主題**:標題下方主題列一鍵點亮整條 AI 鏈 —
  CoWoS 先進封裝、HBM 記憶體、ABF 載板、高速連接器、液冷散熱、AI 網通
  (交換器/光模組)、電力基礎設施(UPS/配電)、CPO 矽光子、ASIC 生態系、
  企業級 SSD 儲存(`data.js` 的 `THEMES` 表);個股面板同步顯示所屬主題
- **搜尋**:代號或名稱(如 `2330`、`聯發科`),自動置中
- **收盤報價**:伺服器代理 TWSE/TPEx 公開 OpenAPI,面板顯示收盤價與漲跌
  (台股慣例紅漲綠跌);抓不到時自動退化成純地圖模式
- **基本面**:EPS(綜合損益表)、本益比、殖利率、淨值比 — 伺服器代理交易所
  開放資料,快取 12 小時
- **產品營收佔比**:主要產品以約略佔比長條呈現(人工整理自年報/法說等
  公開資訊,`data.js` 的 `SHARES` 表,與 `PRODUCTS` 同序對齊)
- **供應線營收貢獻**:重要供應關係標註該客戶約佔供應方營收比重
  (`data.js` 的 `SUPPLY_PCT` 表,約略值),顯示於線上標籤、提示框與面板
- **法說會資訊**:自動同步交易所開放資料(日期、訊息擇要、專區連結)到個股面板
- **研究文件庫**:把法說會逐字稿、券商報告等檔案放進 `docs/<股號>/`,
  面板自動建立索引與下載連結(詳見 `docs/README.md`;券商報告有版權、
  無公開 API,需自行放入)
- **AlphaMemo 逐字稿索引**:`npm run sync:transcripts` 解析
  [AlphaMemo 免費逐字稿](https://www.alphamemo.ai/free-transcripts)索引頁、
  依公司自動配對寫入 `public/transcripts.json`,面板直接列出該股逐字稿連結;
  附 GitHub Actions 每週自動同步(`.github/workflows/sync-transcripts.yml`)
- **外部資源**:每檔個股一鍵連到 MOPS、財報狗、Goodinfo、Yahoo 股市
- **深色模式**:跟隨系統主題;節點依供應鏈上游→下游大致由左至右分布

## 執行

```bash
npm install
npm start          # http://localhost:3000
```

## 資料

- 圖譜資料在 `public/data.js`,為公開資訊之人工整理(2026 年中),
  結構簡單(`NODES` / `LINKS`),直接編輯即可增修個股與關係。
- 報價來源:
  - 上市:`https://openapi.twse.com.tw/v1/exchangeReport/STOCK_DAY_ALL`
  - 上櫃:`https://www.tpex.org.tw/openapi/v1/tpex_mainboard_daily_close_quotes`
  - 伺服器端快取 10 分鐘,失敗時回傳舊快取或空資料。
- 法說會來源(伺服器端快取 6 小時,守備式欄位解析):
  - 上市:`https://openapi.twse.com.tw/v1/opendata/t187ap38_L`
  - 上櫃:`https://www.tpex.org.tw/openapi/v1/mopsfin_t187ap38_O`
- 研究文件:`docs/<股號>/日期_類型_標題.pdf`,由 `/api/docs` 自動掃描建索引。

> 本專案僅供產業研究與學習,不構成投資建議。

## 市場聲量(大家都在看的公開來源)

個股面板的「市場聲量・未經證實」區,由 `npm run sync:rumors`(或每天
09:00/21:00 台北時間的排程)自動同步,寫入 `public/rumors.json`:

- **鉅亨網台股新聞**(免費):公開新聞列表 API,中文標題以公司名精準比對
- **Yahoo 股市新聞**(免費):補海外個股;台股四碼需含中文或 TPE: 標記
  才收,避免他國交易所同號誤歸戶
- **PTT 股票板**(免費,僅本機執行可用):PTT 擋雲端機房 IP,
  GitHub Actions 抓不到;在自己電腦跑 `npm run sync:rumors` 可補上
- **X KOL**(選配):需 X API Basic 的 `X_BEARER_TOKEN` repo secret
  與 `config/kols.json`;未設定時安靜跳過,不做違反服務條款的網頁爬蟲
