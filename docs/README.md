# 研究文件庫

把個股相關的**法說會逐字稿、券商報告、法說簡報**等檔案放進本資料夾,
地圖的個股面板會自動列出並提供下載連結,不需要改任何程式碼。

## 使用方式

1. 依股票代號建資料夾,把檔案丟進去:

```
docs/
├── 2330/
│   ├── 2026-01-16_逐字稿_2025Q4法說會.pdf
│   ├── 2026-01-20_券商報告_外資最新目標價整理.pdf
│   └── 2025-10-17_法說簡報_2025Q3業績說明.pdf
└── 2454/
    └── 2026-02-02_逐字稿_2025Q4法說會.pdf
```

2. 檔名格式:`日期_類型_標題.副檔名`
   - **日期**:`YYYY-MM-DD` 或 `YYYYMMDD`(可省略)
   - **類型**:自由填寫,如 `逐字稿`、`券商報告`、`法說簡報`(可省略,預設「文件」)
   - 支援副檔名:pdf / txt / md / html / doc(x) / ppt(x) / xls(x) / csv
3. 重新整理網頁,個股面板的「研究文件」區就會出現。

## 逐字稿哪裡找?

- **AlphaMemo 免費逐字稿**:<https://www.alphamemo.ai/free-transcripts>
  - **自動索引**:執行 `npm run sync:transcripts`(或等 GitHub Actions 每週自動跑),
    會解析索引頁、依公司自動配對,寫入 `public/transcripts.json`,
    個股面板「研究文件」就會直接列出該股的逐字稿連結(開啟 AlphaMemo 原站)。
    AlphaMemo 的網址是不透明 UUID(如 `/free-transcripts/9fa1db9a-…`),
    無法從股號推導,所以需要這個索引步驟。
  - 想離線收藏的話,下載/整理成 PDF 放進 `docs/<股號>/`,並註明出處。
- **MOPS 法說會專區**:官方簡報檔與影音連結(面板「法說會」區會自動同步場次)。

## 為什麼不自動下載?

- **法說會逐字稿**:台灣官方管道(MOPS)只公告法說會的日期、地點、訊息擇要、
  簡報檔與影音連結,**並不提供逐字稿**;逐字稿多為付費服務或自行整理的產物。
  地圖已自動同步交易所開放資料的法說會場次資訊(個股面板「法說會」區)。
- **券商報告**:屬券商智慧財產,在各券商的會員專區內,**沒有合法的公開 API**
  可以自動抓取。請從你有權限的管道下載後放入本資料夾。

> 提醒:本資料夾內容若含有版權文件,建議將 GitHub repo 設為 private,
> 或只放自己整理的筆記與公開簡報,避免公開散布受版權保護的報告。
